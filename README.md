# 速記練習用の朗読補助アプリ

入力した速度で読むときの読み上げ速さを画面端のバーで表示します。バーが進むのに合わせて音読することで、一定速度を保った読み上げが簡単に行えます。

## バーの上端、下端の位置合わせ

画面上部の歯車マークを押すと、バーの上端、下端をドラッグ操作で自由に移動できるようになります。印刷、表示した問題文の幅に合わせると読みやすいです。

調整後はチェックボタンを押すと位置が固定されます。調整後の情報は端末に保存され、次回アクセス時も同じ幅で表示されます。

## バーの左右変更

画面上部の矢印でバーの左右を変更できます。スマホを持つ位置や利き手に合わせて調節してください。

## ダークモード対応

画面上部の月マーク、太陽マークでダークモード、ライトモードを切り替えることができます。
